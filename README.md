## 🐸 Frog Jump Game

> Python을 공부하려고 게임을 만들기 시작했는데, 만들다 보니 점수를 저장하고 싶어졌어요.

게임을 만들다가 서버가 필요해졌고, 서버를 만들다가 랭킹 페이지가 갖고 싶어졌어요.  
그 다음엔 회원가입·로그인·게시판·포인트 시스템까지 갖춘 **공식 웹 플랫폼**을 직접 만들었어요.  
24살에 이 구조를 혼자 설계하고 배포까지 해봤다는 게 저한테는 꽤 큰 경험이었어요.

**v1.0에서 바뀐 것들**

- 🎨 동글동글한 폰트와 아기자기한 테마 적용
- 🏆 게임 오버 시 메달 시스템이 있는 랭킹 보드 추가
- ⏱ 시간 10초 이하일 때 두근거리는 HUD 애니메이션
- 🧩 `core/` 패키지로 완전히 모듈화

**만들면서 막혔던 것들**

- 🐛 서버 요청이 게임 루프를 막아서 입력 지연이 생겼어요 → Threading으로 해결하면서 동기/비동기가 처음으로 몸으로 와닿았어요
- 🐛 클라이언트 점수와 서버 저장값이 달랐어요 → 데이터 처리 순서를 명시적으로 보장해야 한다는 걸 그때 배웠습니다
