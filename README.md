## 🔥 Main Project

### 🐸 Frog Jump Game (Full Stack + Game + Server)

**✔ 핵심 기능**

* FastAPI 기반 REST API 서버 구축
* 점수 저장 및 **랭킹 시스템 (TOP5) 구현**
* 웹(JavaScript) + 게임(Pygame) 클라이언트 연동

**✔ 기술 포인트**

* HTTP / JSON 기반 클라이언트-서버 통신 구조 설계
* SQLite → PostgreSQL 확장 가능한 DB 구조 설계
* Threading 기반 비동기 점수 처리

**✔ 아키텍처**
Frontend (Vercel) → FastAPI (Render) → Database

**✔ API**

* `POST /score` : 점수 저장
* `GET /leaderboard` : 랭킹 조회
